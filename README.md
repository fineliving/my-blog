# 技术胖的博客系统

> 本套博客采用了VuePress搭建，样式跟进网络开源模板进行修改。

## 博客功能

- 全静态博客，可用Vue来定义模板
- 可以集成github上的评论
- 有文章的目录功能
- 支持分类和tag分组
- 增加了右侧打赏功能
